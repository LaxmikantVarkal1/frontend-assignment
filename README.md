# Frontend Assignment

This project includes the following features:

## Features

1. Simple Calculator
- Adds two numbers and displays result on screen

2. Input Focus Styling
- Changes background color when input field is focused

3. Image Hover Effect
- Scales image to 1.5x on hover using CSS

4. Async Function
- Adds two numbers using Promise
- Handles valid and invalid inputs with proper error handling

## Tech Stack
- HTML
- CSS
- JavaScript
